x402 Live Flow HTTP 402 Protocol

An AI agent calls a paid API. The server returns 402. The SDK automatically signs a payment, retries, and gets the response — all in one payAndFetch() call.

Click ▶ to run the x402 payment flow...
Buyer SDK AI Agent 
import { CyberNanoPayBuyer } from "cyber-nano-pay-sdk";

// Initialize once with your TON address + keypair
const buyer = new CyberNanoPayBuyer({
  keypair: myEd25519Keypair,
  address:    "EQD...",
  gatewayUrl: "https://gateway.cyberpay.dev",
});

// Pay for any 402-protected API — one line
const res = await buyer.payAndFetch(
  "https://api.example.com/ai/generate"
);
const data = await res.json();

// Or sign manually for custom flows
const auth = buyer.signAuthorization({
  to: sellerAddress,  amount: 1000n, // $0.001 USDT
});

import { requirePayment } from "cyber-nano-pay-gateway";

// Gate any endpoint with one middleware line
app.get("/api/ai/generate",
  requirePayment({
    amount: "1000",      // $0.001 USDT per call
    to:     myTonAddress,
  }),
  async (c) => {
    // Payment verified by TEE — run your logic
    const payment = c.get("payment");
    // payment.confirmationId, .from, .amount
    return c.json({ result: "AI response" });
  }
);

// TEE receipt auto-attached — verifiable on-chain
// No payment infra to manage. Zero gas.
